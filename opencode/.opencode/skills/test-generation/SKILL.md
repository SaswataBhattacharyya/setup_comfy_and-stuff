name: test-generation
description: Add or improve tests for existing code using the project's current test framework.
compatibility: opencode
---

## Goal

Improve confidence through targeted tests.

## Workflow

1. Detect the test framework.
2. Inspect existing test style.
3. Identify behavior worth testing.
4. Add focused unit/integration tests.
5. Run the relevant test command.
6. Fix test issues if needed.
7. Summarize coverage added.

## Rules

- Match existing test conventions.
- Avoid brittle implementation-detail tests.
- Prefer small meaningful tests.
