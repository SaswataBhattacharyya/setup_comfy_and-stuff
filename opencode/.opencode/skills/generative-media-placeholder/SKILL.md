## Goal

Prepare prompts, workflows, metadata, and asset plans for future image/video generation tools.

## Workflow

1. Identify task: t2ima, ima2ima, or ima2vid.
2. Define input assets and constraints.
3. Write prompt variants.
4. Define style, composition, motion, camera, lighting, duration, aspect ratio.
5. Save prompt/workflow files if requested.
6. State clearly if generation requires an external API/MCP/tool.

## Rules

- Do not claim to generate images/videos unless a tool is connected.
- Preserve subject identity and user constraints.
