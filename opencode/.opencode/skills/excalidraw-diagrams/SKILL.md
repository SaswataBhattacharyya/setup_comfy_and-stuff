## Goal

Generate clear editable diagrams for planning and communication.

## Workflow

1. Identify diagram type: architecture, flowchart, UI wireframe, data flow, pipeline.
2. Use Excalidraw MCP if enabled.
3. Otherwise create/edit .excalidraw JSON files where appropriate.
4. Keep diagrams readable and grouped.
5. Add labels and arrows clearly.
6. Summarize how to open/edit the diagram.

## Rules

- Prefer editable diagrams over static PNG.
- Keep visual complexity manageable.
