## Goal

Use Blender MCP carefully for 3D scene creation and manipulation.

## Workflow

1. Confirm Blender is open and the Blender MCP addon is running.
2. Confirm blender-mcp is enabled in opencode.json.
3. Break scene creation into small operations.
4. Inspect the scene before modifying it.
5. Save work before risky operations.
6. Summarize camera, lights, objects, and materials changed.

## Rules

- Do not run arbitrary Python without approval.
- Do not download external assets without approval.
- Keep Blender MCP disabled unless actively used.
