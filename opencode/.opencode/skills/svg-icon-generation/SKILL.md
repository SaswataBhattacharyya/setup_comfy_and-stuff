## Goal

Use clean, consistent, accessible SVG/icons.

## Workflow

1. Check existing icon library.
2. Use Animotion MCP if available for real icon search.
3. Prefer established icon sets over random generated SVG.
4. Normalize viewBox, size, stroke, fill, and naming.
5. Create React components when useful.
6. Preserve accessibility.

## Rules

- Avoid huge inline SVG unless necessary.
- Do not use unlicensed assets.
- Keep icons visually consistent.
